                                              Online Train Ticket Booking Management 

    Project Description

This project is a basic Online Train Ticket Booking System developed in C. It allows the user to enter journey details such as starting station, destination, travel date, travel class, train name, and number of passengers. Based on the selected options, the program calculates the total fare and displays a complete booking summary.
This project helps beginners understand user input, conditional statements, data handling, and logical operations in C programming.

    Instructions

1. Open the program in any C compiler (Turbo C, Code::Blocks, VS Code, etc.).

2. Compile and run the program.

3. Enter the required journey details when prompted:

    i. From and To locations

    ii. Date of travel

    iii. Train name and travel class

    iv. Number of passengers

4. The program calculates the fare based on the choices made.

5. A final booking summary with total cost is displayed on the screen.

6. Close the program after viewing the result.
